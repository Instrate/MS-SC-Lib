# Running project

The project is not meant to be run directly, but if there's the reason, you're able to run it in to ways

## **Docker** approach

- Compose and run

```bash
docker-compose up -d -build
```

- Stop and remove

```bash
docker-compose down --rmi all -v
```

## Native approach

- Initialize

```bash
npm i
```

- Run

```bash
npm run start
```

- Build

```bash
npm run build
```
